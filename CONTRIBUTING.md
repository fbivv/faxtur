# Contribuer à Faxtur

Merci de proposer les changements par pull request.

Règles importantes :

1. Ne pas modifier `engine/engine_v1.py` sans justification et test de conformité.
2. Toute modification touchant au PDF/A, XMP ou XML Factur-X doit être validée avec veraPDF et un validateur Factur-X.
3. Les fichiers Python doivent conserver l'en-tête MPL 2.0.
4. Les données utilisateur, journaux et dossiers de travail ne doivent pas être versionnés.
