# Installation Windows

## Installation standard

Lancer :

```bat
INSTALLER_FAXTUR.bat
```

Faxtur s'installe dans :

```text
%LOCALAPPDATA%\Faxtur
```

Cette installation ne nécessite pas les droits administrateur.

## Dossiers métier

Par défaut, Faxtur propose des dossiers sur le Bureau de l'utilisateur courant :

- `Factures`
- `Facture-X`
- `A traiter`

Ces chemins peuvent être modifiés dans l'interface.
