# Validation Factur-X

Avant publication d'une release, tester au moins une facture de référence.

Contrôles attendus :

- XML embarqué : OK
- XMP Factur-X : OK
- AFRelationship : OK
- Profil : BASIC WL
- PDF/A : conforme PDF/A-3b
- veraPDF : 0 règle échouée

La validation externe peut être faite avec FactPulse ou un validateur Factur-X équivalent.
